# JUnit Assertions Exercise

## Objective
In this exercise, I used standard JUnit assertions. It tests variables and object state verification.

## Implementation
### Project Files
* `AssertionsTest.java` – Tests verifying various JUnit assertion checks
* `pom.xml` – Maven file containing dependencies and build settings

### Implementation Details
I implemented various assertions such as assertEquals, assertTrue, and assertNotNull to evaluate conditions.

## How to Run
```bash
mvn clean test
```

## Expected Output
The tests check the assertions and returns a successful build report.

### Key Takeaways
I learned how to choose the right JUnit assertion method to test expected values.
