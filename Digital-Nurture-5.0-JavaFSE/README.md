# Digital Nurture 5.0 - Java Full Stack Engineer

This repository contains the hands-on exercises I completed during the Digital Nurture 5.0 Java Full Stack Engineer training program.

I have organised the exercises week by week so they are easier to understand, run, and review.

## Topics Covered

* Design Patterns
* Data Structures and Algorithms
* PL/SQL
* JUnit
* Mockito
* SLF4J Logging
* Maven
* Spring Core
* Spring Data JPA
* Hibernate
* Spring Boot
* Spring REST
* JWT Authentication

## Folder Structure

```text

```

Each exercise folder contains the source code, configuration files, and a README with basic information about the task.

## How to Run the Projects

For Maven projects, open the required exercise folder and run:

```bash
mvn clean test
```

For Spring Boot projects, run:

```bash
mvn spring-boot:run
```

Some programs can also be run directly using their main class from an IDE.
